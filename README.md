# `.github` — Organización VELAR

Este repositorio especial alimenta la **portada pública** de la organización
[Velar-Bonds](https://github.com/Velar-Bonds) y centraliza configuración compartida:

| Ruta | Para qué |
|---|---|
| [`profile/README.md`](profile/README.md) | Portada que se muestra en github.com/Velar-Bonds |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Cómo contribuir a cualquier repo de VELAR |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Código de conducta de la comunidad |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | Plantillas de issues compartidas |
| [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) | Plantilla de PR compartida |

> Estos defaults aplican a todos los repos de la org que no definan los suyos.
